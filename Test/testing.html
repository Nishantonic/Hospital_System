1. <h1> - Heading Tag

Definition: The <h1> tag defines the most important heading. There are six levels of headings, from <h1> to <h6>, with <h1> being the highest.

Syntax: <h1>Your Heading Text</h1>

Example:

<h1>Welcome to My Website</h1>

Welcome to My Website

2. <p> - Paragraph Tag

Definition: The <p> tag defines a paragraph in HTML.

Syntax: <p>Your paragraph text goes here</p>

Example:

<p>This is a paragraph of text.</p>

This is a paragraph of text.

3. <a> - Anchor Tag

Definition: The <a> tag defines a hyperlink, used to link to another page or location.

Syntax: <a href="URL">Link Text</a>

Example:

<a href="https://www.example.com">Visit Example</a>

Example : Anchor Tag

4. <img> - Image Tag

Definition: The <img> tag is used to embed an image in an HTML page.

Syntax: <img src="image.jpg" alt="description" />

Example:

<img src="image.jpg" alt="Sample Image" />

5. <ul> and <li> - List Tags

Definition: The <ul> tag defines an unordered list, and the <li> tag defines a list item.

Syntax: <ul><li>Item</li></ul>

Example:

<ul><li>First Item</li><li>Second Item</li></ul>
  • First Item
  • Second Item

6. <table> - Table Tag

Definition: The <table> tag defines a table in HTML.

Syntax: <table><tr><td>Cell</td></tr></table>

Example:

<table><tr><td>Row 1, Cell 1</td></tr></table>
Row 1, Cell 1 Row 1, Cell 2 Row 1, Cell 3
Row 2, Cell 1 Row 2, Cell 2 Row 2, Cell 3
Row 3, Cell 1 Row 3, Cell 2 Row 3, Cell 3

7. <strong> - Bold Text Tag

Definition: The <strong> tag is used to make text bold and semantically strong.

Syntax: <strong>Bold Text</strong>

Example:

Definition: The <i> tag is used to display text in italics.

Syntax: <i>Italic Text</i>

9. <input> - Input Tag

Definition: The <input> tag is used to create input fields in forms.

Syntax: <input type="text" name="username" />

Definition: The <button> tag defines a clickable button.

Syntax: <button>Button Text</button>
